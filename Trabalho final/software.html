GERENCIADOR DE FROTAS V1.1

Um sistema simples criado para auxiliar no controle de veiculos, manutenção e movimentos de uma empresa de transportes

SISTEMA

TELA LOGIN

O usuario, na primeira vez que utilizar o sistema, entrará com ambos usuario e senha: admin, quando for detectado um registro na tabela usuarios, a conta admin não estará mais disponivel

CADASTRO DE EMPRESA

Nessa tela, o usuario poderá cadastrar a empresa no sistema para usar como ficha cadastral futuramente

LISTA DE FUNCIONARIOS

Aqui os funcionarios serão listados por CPF, Nome e Cargo. Ao clicar em incluir, abrirá uma nova janera para registrar os funcionarios com mais informações

CADASTRO FUNCIONARIOS

Janela para incluir e editar os funcionarios da empresa

MANUTENÇÕES

Tela para registrar as manutenções nos veiculos da empresa

MOVIMENTOS

Tela para registrar os movimentos do mês da empresa e gerar um relatorio dos gastos

RELATORIOS

Uma breve amostra de como ficará o relatorio dos movimentos

VEICULOS

Tela para registrar os veiculos da empresa

USUARIOS

Tela para registrar os usuarios que acessarão o sistema
